Tienda de Ropa Backend

API RESTful

Código fuente en https://github.com/alvarodelmoral/ropaBackend

Se proporciona la siguiente API:

PRENDAS

/api/prenda    POST
Crear prenda
/api/prenda    GET ALL
Obtener todas las prendas.
/api/prenda/id    GET
Obtener prenda
/api/prenda/id    PUT
Modificar prenda
/api/prenda/id    DELETE
Eliminar prenda

CLIENTES

/api/clientes    POST
Crear cliente
/api/clientes    GET ALL
Obtener todos los clientes.
/api/clientes/id    GET
Obtener cliente
/api/clientes/id    PUT
Modificar cliente
/api/clientes/id    DELETE
Eliminar cliente

Frontend en https://ropafrontend.herokuapp.com
